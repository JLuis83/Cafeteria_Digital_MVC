# Cafetería Digital MVC

Proyecto desarrollado en Java aplicando el patrón de arquitectura **MVC (Modelo - Vista - Controlador)**.

Simula la gestión de una cafetería mediante consola (CLI), permitiendo administrar productos, pedidos y pagos.

---

## Tecnologías utilizadas

- Maven
- Arquitectura MVC
- Colecciones Java (ArrayList, HashMap, TreeMap)

---
